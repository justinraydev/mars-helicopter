# three.js webpack

Started project for three.js. Configured with webpack as a bundler.

## Development

Clone the project and install dependencies:

```bash
git clone https://github.com/aakatev/three-js-webpack.git
npm i
```

Start webpack in watch mode

```bash
npm run build:dev
```

Start a server

```bash
npm run serve
```

## Bundling

Run prettier

```bash
npm run format
```

Bundle your code

```bash
npm run build
```
